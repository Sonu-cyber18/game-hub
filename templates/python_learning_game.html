Variables and Data Types

In Python, variables are used to store data values. Unlike some other programming languages, Python has no command for declaring a variable.

A variable is created the moment you first assign a value to it.

# Variable assignment examples
name = "Alice"
age = 30
height = 5.9
is_student = True

Python has various data types: integers, floats, strings, and booleans.
